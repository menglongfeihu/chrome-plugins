# 快恢复

Chrome 扩展，记录并快速恢复最近关闭的标签页。

## 功能

### 弹窗

- 展示关闭历史，可选显示网站图标、页面网址、关闭时间
- 搜索：按标题、网址或两者搜索，支持大小写敏感
- 分页浏览：首页、上一页、下一页、末页、跳转到指定页
- 全选 / 反选当前搜索结果
- 批量打开选中的标签页
- 清空全部 / 清空选中
- 单条从列表移除
- 点击条目直接恢复标签页

### 设置页

**界面**

| 配置项 | 说明 |
|--------|------|
| 显示宽度 | 弹窗宽度，300–600 px |
| 显示高度 | 弹窗高度，200–800 px |
| 最多保存条数 | 历史上限，5–500 条 |
| 每页大小 | 分页每页条数，10–50 条，步长 5 |
| 显示时间 | 是否展示关闭时间 |
| 显示网站图标 | 是否展示 favicon |
| 显示页面网址 | 是否展示 URL |
| 图标显示数量 | 是否在扩展图标上显示未恢复条数 |
| 主题 | 跟随系统 / 亮色 / 暗色 |

**功能**

| 配置项 | 说明 |
|--------|------|
| 自定义热键 | 跳转到 Chrome 快捷键设置页 |
| 搜索范围 | 标题 / 网址 / 两者 |
| 大小写敏感搜索 | 默认关闭 |
| 恢复后关闭弹窗 | 点击条目恢复后自动关闭弹窗 |
| 恢复后从列表移除 | 点击条目恢复后自动从历史中删除 |
| 相同网址去重 | 只保留同一网址最近一次关闭记录 |

**屏蔽**

- 添加域名或关键词，匹配的页面关闭时不被记录

**清理**

- 清空全部历史
- 清空 N 天前的历史（1–30 天，滑块选择）

**数据管理**

- 恢复所有设置为默认值

## 安装

1. 打开 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"，选择本目录

## 权限

- `tabs`：监听标签页关闭事件，获取标签标题和 URL
- `storage`：本地存储历史记录和设置
