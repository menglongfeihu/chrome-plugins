# chrome-plugins

Chrome 浏览器插件集合。

---

## 快恢复（chrome-easy-undo）

快速查看和恢复最近关闭的标签页。

### 功能

**弹窗**

- 列出所有最近关闭的标签页，显示标题、网址、网站图标、关闭时间
- 点击条目即可在新标签页中恢复
- 勾选多条后点击"打开选中"批量恢复
- 悬停条目右侧的 × 可单独删除该记录
- 顶部搜索框实时过滤，支持搜索标题 / 网址 / 两者
- 一键清空全部记录
- 图标上的数字徽章实时显示记录数量

**设置页（右下角"设置"按钮进入）**

| 分组 | 配置项 |
|------|--------|
| 界面 | 弹窗宽高、最多保存条数（5–500，默认 300）、显示时间 / 网站图标 / 网址 / 数量徽章、主题（跟随系统 / 亮色 / 暗色） |
| 清理 | 清空全部记录、清理 n 天前的记录 |
| 功能 | 自定义热键、搜索范围（标题 / 网址 / 两者）、大小写敏感搜索、恢复后关闭弹窗、恢复后从列表移除、相同网址去重 |
| 屏蔽 | 配置不被记录的域名或关键词 |
| 数据管理 | 恢复所有设置为默认值 |

### 安装

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"，选择 `chrome-easy-undo` 目录
