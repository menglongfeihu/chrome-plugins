## 快恢复（chrome-easy-undo）

> 详细文档见 [chrome-easy-undo/README.md](chrome-easy-undo/README.md)

Chrome 扩展，记录并快速恢复最近关闭的标签页。

### 主要功能

- 分页浏览关闭历史，支持首页 / 末页 / 跳转，每页条数可配置
- 实时搜索（标题 / 网址 / 两者），全选 / 反选，批量打开或清空
- 设置页配置弹窗尺寸、保存上限、主题（亮色 / 暗色 / 跟随系统）、搜索范围、去重、屏蔽规则等

### 安装

1. 打开 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"，选择 `chrome-easy-undo` 目录
